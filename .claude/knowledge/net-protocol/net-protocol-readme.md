## 选型速查

### RDMA 方案选型

| 特性 | InfiniBand | RoCE v2 | UEC |
|------|-----------|---------|-----|
| 物理层 | IB 专用 | 以太网 | 以太网 |
| 传输层 | IB 可靠传输 | UDP + BTH | UEC 传输层 |
| 延迟 | ~1μs | ~1.5μs | <1μs（目标） |
| 多路径 | 不支持 | 不支持（ECMP 有限） | 原生支持 |
| 拥塞控制 | Credit | ECN+DCQCN | Credit+ECN+RTT |
| 生态 | 成熟（HPC） | 广泛（云厂商） | 新兴（AI 集群） |
| 适用 | 传统 HPC | 当前 AI/云 | 下一代 AI |

### 隧道协议选型

| 特性 | VXLAN | GENEVE | NVGRE |
|------|-------|--------|-------|
| Header 长度 | 8B 固定 | 8B + 可变 TLV | 8B 固定 |
| 可扩展性 | 有限 | 强（TLV） | 有限 |
| VNI 空间 | 24-bit (16M) | 24-bit (16M) | 24-bit (16M) |
| 硬件支持 | 最广泛 | 逐步增加 | 有限 |
| 标准 | RFC 7348 | RFC 8926 | RFC 7637 |
| 推荐 | 通用首选 | 需要可扩展性时 | 兼容旧设备 |

## 文档格式说明

每个协议文档包含以下标准章节：

| 章节 | 内容 |
|------|------|
| 协议概述 | 特性、定位、规范版本、典型应用 |
| 报文/帧格式 | Header ASCII 框图，逐字段位宽说明 |
| 核心机制 | 传输流程、仲裁、流控、错误处理等 |
| 时序/流程 | ASCII 波形或事务流程图 |
| 对比表 | 与同类协议的特性对比 |
| 设计注意事项 | NIC/交换芯片 RTL 设计要点，硬件加速方案 |

**配合 chip-arch agent 使用**：在生成 FS/微架构文档时，agent 会检索本目录下的协议文档作为接口设计参考。
