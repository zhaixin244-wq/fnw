# 内存属性

> **用途**：处理器内存属性设计参考，供 CPU/SoC 缓存子系统设计时检索
> **典型应用**：所有支持缓存的系统

---

## 概述

内存属性定义了内存区域的缓存策略、共享性和访问特性。

### 内存属性类型

| 类型 | 说明 | 典型应用 |
|------|------|----------|
| Cache 属性 | 缓存策略（WB/WT/UC） | 所有支持缓存的系统 |
| 共享性 | 多核共享策略 | 多核系统 |
| 内存类型 | 内存访问特性 | 所有系统 |

---

## RISC-V 内存属性

### PMA（物理内存属性）

```
PMA 定义物理内存区域的属性：

┌─────────────────────────────────────────────┐
│  PMA 区域                                   │
│  ┌──────────────────────────────────────┐   │
│  │  地址范围                            │   │
│  │  Cache 属性                          │   │
│  │  访问特性                            │   │
│  └──────────────────────────────────────┘   │
└─────────────────────────────────────────────┘
```

### PMA Cache 属性

| 属性 | 说明 | 缓存行为 |
|------|------|----------|
| Cacheable | 可缓存 | 写回/写直达 |
| Non-cacheable | 不可缓存 | 直接访问内存 |
| Write-back | 写回 | 写入缓存，延迟写回 |
| Write-through | 写直达 | 同时写入缓存和内存 |

### PTE Cache 属性

```
RISC-V PTE 没有直接的 Cache 属性字段，
Cache 属性由 PMA 或平台定义。
```

## ARM 内存属性

### MAIR_ELx（内存属性间接寄存器）

```
MAIR_ELx:
┌─────────────────────────────────────────────┐
│  Attr0  Attr1  Attr2  Attr3  Attr4  Attr5  Attr6  Attr7 │
│  8-bit  8-bit  8-bit  8-bit  8-bit  8-bit  8-bit  8-bit │
└─────────────────────────────────────────────┘
```

### 属性编码

| Attr[7:4] | Attr[3:0] | 类型 | 说明 |
|-----------|-----------|------|------|
| 0000 | 0000 | Device-nGnRnE | 设备内存，最严格 |
| 0000 | 0001 | Device-nGnRE | 设备内存，宽松 |
| 0000 | 0010 | Device-GRE | 设备内存，最宽松 |
| 0000 | 0100 | Device | 设备内存 |
| 0100 | 0000 | Normal NC | 普通内存，非缓存 |
| 1010 | 1010 | Normal WT | 普通内存，写直达 |
| 1110 | 1110 | Normal WB | 普通内存，写回 |
| 1111 | 1111 | Normal WB RW-alloc | 普通内存，写回，读写分配 |

### PTE 内存属性

```
PTE:
┌─────┬─────┬─────┬─────┬─────┬─────┐
│     │     │     │     │ Attr│     │
│     │     │     │     │ Indx│     │
└─────┴─────┴─────┴─────┴─────┴─────┘

AttrIndx: 指向 MAIR_ELx 的索引（0-7）
```

## x86 内存属性

### 页表属性

```
PTE:
┌─────┬─────┬─────┬─────┬─────┬─────┬─────┬─────┐
│  PAT│     │     │     │  PCD│  PWT│     │     │
└─────┴─────┴─────┴─────┴─────┴─────┴─────┴─────┘

PAT: Page Attribute Table 选择
PCD: Page Cache Disable
PWT: Page Write-Through
```

### PAT 组合

| PAT | PCD | PWT | 内存类型 |
|-----|-----|-----|----------|
| 0 | 0 | 0 | WB (Write-Back) |
| 0 | 0 | 1 | WT (Write-Through) |
| 0 | 1 | 0 | UC- (Uncached) |
| 0 | 1 | 1 | UC (Uncached) |
| 1 | 0 | 0 | WC (Write-Combining) |
| 1 | 1 | 1 | WP (Write-Protected) |

## 缓存策略

### 写回（Write-Back）

```
CPU 写入 → 写入缓存 → 延迟写回内存

优点：
- 写延迟低
- 总线流量小
- 能效高

缺点：
- 脏数据管理复杂
- 一致性协议复杂
```

### 写直达（Write-Through）

```
CPU 写入 → 同时写入缓存和内存

优点：
- 一致性简单
- 数据始终最新

缺点：
- 写延迟高
- 总线流量大
```

### 非缓存（Uncached）

```
CPU 写入 → 直接写入内存

优点：
- 数据始终最新
- 无一致性问题

缺点：
- 访问延迟高
- 性能低
```

### 写合并（Write-Combining）

```
多个写入 → 合并为一次内存访问

优点：
- 减少内存访问次数
- 提高带宽利用率

缺点：
- 写延迟不确定
- 需要特殊硬件支持
```

## 共享性

### ARM 共享性

```
PTE:
┌─────┬─────┬─────┬─────┬─────┬─────┐
│     │     │     │ SH  │ SH  │     │
│     │     │     │ [1] │ [0] │     │
└─────┴─────┴─────┴─────┴─────┴─────┘

SH[1:0]:
  00: Non-shareable
  01: Reserved
  10: Outer Shareable
  11: Inner Shareable
```

### 共享性类型

| 类型 | 说明 | 典型应用 |
|------|------|----------|
| Non-shareable | 不共享 | 单核系统 |
| Inner Shareable | 内部共享 | 同一集群内共享 |
| Outer Shareable | 外部共享 | 跨集群共享 |

### 共享性与缓存一致性

- **Non-shareable**：无需一致性协议
- **Inner Shareable**：需要内部一致性协议
- **Outer Shareable**：需要外部一致性协议

## 设备内存

### 设备内存类型

| 类型 | 说明 | 典型应用 |
|------|------|----------|
| Device-nGnRnE | 最严格 | 寄存器访问 |
| Device-nGnRE | 宽松 | 设备内存 |
| Device-GRE | 最宽松 | 设备内存 |

### 设备内存特性

- **不可缓存**：设备内存不可缓存
- **不可合并**：写操作不可合并
- **顺序保证**：保证访问顺序
- **副作用**：访问可能有副作用

## 内存类型选择

### 选择指南

| 场景 | 内存类型 | 说明 |
|------|----------|------|
| 代码段 | Normal WB | 缓存提高性能 |
| 数据段 | Normal WB | 缓存提高性能 |
| 栈 | Normal WB | 缓存提高性能 |
| DMA 缓冲区 | Normal NC | 设备访问一致性 |
| 设备寄存器 | Device-nGnRnE | 保证访问顺序 |
| 帧缓冲区 | WC | 写合并提高性能 |

### 常见错误

1. **设备内存缓存**：设备内存不可缓存
2. **DMA 缓冲区缓存**：DMA 缓冲区需非缓存或一致性
3. **内存类型不一致**：同一物理页不同内存类型

## 设计注意事项

### RTL 实现要点

1. **属性检查**：需在内存访问时检查属性
2. **缓存行为**：需根据属性控制缓存行为
3. **共享性**：需根据属性控制一致性
4. **设备内存**：需保证设备内存访问顺序
5. **性能优化**：需优化常用内存类型的访问

### 面积/功耗权衡

| 配置 | 面积 | 功耗 | 说明 |
|------|------|------|------|
| 简单缓存 | 小 | 低 | 只支持 WB |
| 复杂缓存 | 中 | 中 | 支持多种属性 |
| 完整缓存 | 大 | 高 | 支持所有属性 |

### 常见陷阱

1. **内存类型不一致**：同一物理页不同内存类型
2. **设备内存缓存**：设备内存不可缓存
3. **DMA 一致性**：DMA 缓冲区需特殊处理
4. **共享性配置**：共享性配置错误导致一致性问题
5. **属性检查**：需在内存访问时检查属性
