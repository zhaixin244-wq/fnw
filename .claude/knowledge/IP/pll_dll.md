# PLL/DLL IP

> **用途**：时钟生成 IP 选型参考，供 SoC 时钟系统设计时检索
> **典型应用**：所有需要时钟的系统

---

## 概述

PLL（Phase-Locked Loop）和 DLL（Delay-Locked Loop）是时钟生成和管理的关键组件。

### PLL vs DLL

| 特性 | PLL | DLL |
|------|-----|-----|
| 原理 | 相位锁定 | 延迟锁定 |
| 频率 | 可倍频/分频 | 仅分频 |
| 抖动 | 较高 | 较低 |
| 面积 | 较大 | 较小 |
| 应用 | 时钟生成 | 时钟对齐 |

---

## PLL 架构

### 基本 PLL 架构

```
┌─────────────────────────────────────────────┐
│  PLL                                        │
│  ┌──────────────────────────────────────┐   │
│  │  相位检测器 (PD)                      │   │
│  │  比较参考时钟和反馈时钟相位           │   │
│  └──────────────────────────────────────┘   │
│  ┌──────────────────────────────────────┐   │
│  │  低通滤波器 (LPF)                     │   │
│  │  滤除高频噪声                         │   │
│  └──────────────────────────────────────┘   │
│  ┌──────────────────────────────────────┐   │
│  │  压控振荡器 (VCO)                     │   │
│  │  生成输出时钟                         │   │
│  └──────────────────────────────────────┘   │
│  ┌──────────────────────────────────────┐   │
│  │  分频器 (DIV)                         │   │
│  │  反馈分频和输出分频                   │   │
│  └──────────────────────────────────────┘   │
└─────────────────────────────────────────────┘
```

### PLL 参数

| 参数 | 说明 | 典型值 |
|------|------|--------|
| 输入频率 | 参考时钟频率 | 10-100 MHz |
| 输出频率 | VCO 输出频率 | 100 MHz-3 GHz |
| 分频比 | N = f_out / f_in | 1-1024 |
| 锁定时间 | PLL 锁定时间 | 1-100 μs |
| 抖动 | 输出时钟抖动 | 1-100 ps |
| 功耗 | PLL 功耗 | 1-100 mW |

┌─────────────────────────────────────────────┐

└──────────────────┬──────────────────────────┘
                   │
┌──────────────────▼──────────────────────────┐
│  PLL                                        │
│  (倍频到高频)                                │
└──────────────────┬──────────────────────────┘
                   │

│  时钟分频器                                  │
│  (生成多种频率)                              │
└──────────────────┬──────────────────────────┘
                   │
    ┌──────────────┼──────────────┐
    │              │              │
┌───▼───┐    ┌────▼────┐    ┌────▼────┐
│ CPU   │    │  AXI    │    │  外设   │

## 选型建议

### 嵌入式

- **推荐**：ARM PLL
- **配置**：100 MHz-500 MHz，低功耗
- **理由**：面积小，功耗低

### 通用 SoC

- **推荐**：Synopsys PLL
- **配置**：100 MHz-2 GHz，多输出
- **理由**：灵活配置，性能好

### 高速接口

- **推荐**：Synopsys/Cadence PLL
- **配置**：1-5 GHz，低抖动
- **理由**：高频，低抖动

### DDR 接口

- **推荐**：Synopsys/Cadence DLL
- **配置**：DDR 频率，低抖动
- **理由**：与 DDR 时序匹配

## 集成注意事项

### 接口匹配

- **参考时钟**：确保参考时钟稳定
- **电源**：PLL/DLL 对电源敏感
- **地**：良好的接地设计

### 验证策略

- **功能验证**：PLL/DLL 锁定功能
- **性能验证**：频率、抖动、锁定时间
- **兼容性**：与系统时钟兼容

### 物理集成

- **布局**：PLL/DLL 布局约束
- **电源**：独立电源域
- **屏蔽**：时钟信号屏蔽
