# 以太网 IP

> **用途**：以太网 MAC/PHY IP 选型参考，供网卡/交换芯片/SoC 架构设计时检索
> **典型应用**：网卡、交换芯片、SoC、DPU

---

## 概述

以太网是最广泛使用的局域网技术，以太网 IP 包括 MAC（媒体访问控制）和 PHY（物理层）。

## 以太网架构

### MAC 架构

```
┌─────────────────────────────────────────────┐
│  以太网 MAC                                 │
│  ┌──────────────────────────────────────┐   │
│  │  AXI 接口                            │   │
│  │  (接收/发送数据)                     │   │
│  └──────────────────────────────────────┘   │
│  ┌──────────────────────────────────────┐   │
│  │  TX FIFO                             │   │
│  │  (发送缓冲)                          │   │
│  └──────────────────────────────────────┘   │
│  ┌──────────────────────────────────────┐   │
│  │  RX FIFO                             │   │
│  │  (接收缓冲)                          │   │
│  └──────────────────────────────────────┘   │
│  ┌──────────────────────────────────────┐   │
│  │  MAC 控制器                          │   │
│  │  (帧封装/解封)                       │   │
│  └──────────────────────────────────────┘   │
│  ┌──────────────────────────────────────┐   │
│  │  MII/GMII/XGMII 接口                │   │
│  │  (连接 PHY)                          │   │
│  └──────────────────────────────────────┘   │
└─────────────────────────────────────────────┘
```

### PHY 架构

```
┌─────────────────────────────────────────────┐
│  以太网 PHY                                 │
│  ┌──────────────────────────────────────┐   │
│  │  PCS (物理编码子层)                   │   │
│  │  (编码/解码)                         │   │
│  └──────────────────────────────────────┘   │
│  ┌──────────────────────────────────────┐   │
│  │  PMA (物理媒介附加子层)               │   │
│  │  (串行/解串)                         │   │
│  └──────────────────────────────────────┘   │
│  ┌──────────────────────────────────────┐   │
│  │  PMD (物理媒介相关子层)               │   │
│  │  (驱动/接收)                         │   │
│  └──────────────────────────────────────┘   │
│  ┌──────────────────────────────────────┐   │
│  │  MDI (媒介相关接口)                   │   │
│  │  (连接物理媒介)                      │   │
│  └──────────────────────────────────────┘   │
└─────────────────────────────────────────────┘
```

## 以太网配置选项

### MAC 配置

| 配置 | 选项 | 权衡 |
|------|------|------|
| 速率 | 1G/2.5G/10G/25G/100G | 性能 vs 功能 |
| 接口 | MII/GMII/XGMII/USXGMII | 兼容性 vs 性能 |
| 缓冲 | FIFO 大小 | 性能 vs 面积 |
| 功能 | 基本/完整 | 功能 vs 面积 |

### PHY 配置

| 配置 | 选项 | 说明 |
|------|------|------|
| 速率 | 1G/10G/25G | 与 MAC 匹配 |
| 媒介 | 铜缆/光纤 | 物理接口 |
| SerDes | 内置/外部 | 高速串行 |
| 校准 | 支持/不支持 | 时序校准 |

## 选型建议

### 嵌入式

- **推荐**：Synopsys 1G MAC
- **配置**：1G，MII 接口
- **理由**：面积小，功耗低

### 企业网

- **推荐**：Synopsys 2.5G/5G MAC
- **配置**：2.5G/5G，USXGMII 接口
- **理由**：兼容现有基础设施

### 服务器

- **推荐**：Synopsys/Cadence 10G/25G MAC/PCS
- **配置**：10G/25G，XGMII/USXGMII 接口
- **理由**：高带宽，功能完整

### 数据中心

- **推荐**：Synopsys/Cadence 100G/200G/400G
- **配置**：100G+，多通道
- **理由**：极高带宽

## 集成注意事项

### 接口匹配

- **AXI 接口**：确保与 SoC 互联匹配
- **时钟域**：以太网时钟与系统时钟同步
- **中断**：接收/发送中断

### 验证策略

- **协议验证**：以太网协议一致性
- **性能验证**：带宽和延迟测试
- **兼容性**：与其他以太网设备兼容

### 物理集成

- **SerDes**：高速 SerDes 布局
- **电源**：独立电源域
- **DFT**：扫描链、BIST 集成
