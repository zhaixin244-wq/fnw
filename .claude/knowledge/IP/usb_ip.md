# USB IP

> **用途**：USB 控制器 IP 选型参考，供消费电子/SoC 架构设计时检索
> **典型应用**：消费电子、IoT、嵌入式

---

## 概述

USB（Universal Serial Bus）是最广泛使用的外设接口标准。

### USB 版本对比

| 版本 | 速率 | 典型应用 |
|------|------|----------|
| USB 1.1 | 12 Mbps | 旧设备 |
| USB 2.0 | 480 Mbps | 通用外设 |
| USB 3.0 | 5 Gbps | 高速存储 |
| USB 3.1 | 10 Gbps | 高速存储 |
| USB 3.2 | 20 Gbps | 高速存储 |
| USB4 | 40 Gbps | 高速接口 |
| USB4 2.0 | 80 Gbps | 未来高速接口 |

### 带宽对比

| 版本 | 理论带宽 | 有效带宽 |
|------|----------|----------|
| USB 2.0 | 480 Mbps | 35 MB/s |
| USB 3.0 | 5 Gbps | 400 MB/s |
| USB 3.1 | 10 Gbps | 800 MB/s |
| USB 3.2 | 20 Gbps | 1.6 GB/s |
| USB4 | 40 Gbps | 3.2 GB/s |

│  ┌──────────────────────────────────────┐   │

## USB PHY

### PHY 接口

| 接口 | 版本 | 速率 | 典型应用 |
|------|------|------|----------|
| UTMI | USB 2.0 | 480 Mbps | 嵌入式 |
| ULPI | USB 2.0 | 480 Mbps | 移动设备 |
| PIPE | USB 3.0+ | 5+ Gbps | 高速接口 |

### PHY 功能

- **时钟恢复**：从数据中恢复时钟
- **串行/解串**：并串转换
- **编码/解码**：8b/10b 或 128b/130b
- **驱动/接收**：差分信号驱动

## 选型建议

### 嵌入式

- **推荐**：Synopsys USB 2.0 Device
- **配置**：USB 2.0，Device 模式，4 端点
- **理由**：面积小，功耗低

### 消费电子

- **推荐**：Synopsys USB 3.0 Host/Device
- **配置**：USB 3.0，Host/Device 模式，8 端点
- **理由**：高速，支持多种外设

### 移动设备

- **推荐**：Synopsys USB 3.1 OTG
- **配置**：USB 3.1，OTG 模式，充电支持
- **理由**：高速，支持充电

### 服务器

- **推荐**：Synopsys USB 3.0 Host
- **配置**：USB 3.0，Host 模式，调试支持
- **理由**：高速，支持调试

## 集成注意事项

### 接口匹配

- **AXI 接口**：确保与 SoC 互联匹配
- **时钟域**：USB 时钟与系统时钟同步
- **中断**：传输完成中断

### 验证策略

- **协议验证**：USB 协议一致性
- **性能验证**：带宽和延迟测试
- **兼容性**：与 USB 设备兼容

### 物理集成

- **PHY 布局**：高速 PHY 布局约束
- **电源**：独立电源域
- **DFT**：扫描链、BIST 集成
