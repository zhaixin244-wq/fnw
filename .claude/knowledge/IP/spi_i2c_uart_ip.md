# SPI/I2C/UART IP

> **用途**：低速外设 IP 选型参考，供嵌入式/SoC 架构设计时检索
> **典型应用**：嵌入式、IoT、传感器接口

---

## 概述

SPI、I2C、UART 是最常用的低速外设接口。

### 接口对比

| 接口 | 速率 | 线数 | 通信方式 | 典型应用 |
|------|------|------|----------|----------|
| SPI | 1-100+ MHz | 4 | 同步全双工 | Flash、传感器 |
| I2C | 100k-3.4 MHz | 2 | 同步半双工 | EEPROM、传感器 |
| UART | 9600-3M bps | 2 | 异步全双工 | 调试串口 |

---

## SPI IP

### SPI 特性

- **速率**：1-100+ MHz
- **线数**：4 线（SCLK、MOSI、MISO、CS）
- **通信**：同步全双工
- **主从**：支持主/从模式

### SPI IP 供应商

| 供应商 | IP | 特性 | 典型应用 |
|--------|------|------|----------|
| Synopsys | DesignWare SPI | 主/从、DMA | 通用 |
| Cadence | Cadence SPI | 主/从、DMA | 通用 |
| ARM | ARM SPI | 主模式 | 嵌入式 |

### SPI 配置

| 配置 | 选项 | 说明 |
|------|------|------|
| 模式 | 主/从/主从 | 工作模式 |
| 速率 | 1-100 MHz | 时钟频率 |
| 数据宽度 | 4-32 bit | 数据位数 |
| CS 数量 | 1-16 | 片选数量 |
| DMA | 支持/不支持 | 数据传输 |
| FIFO | 深度 | 缓冲大小 |

---

## I2C IP

### I2C 特性

- **速率**：100k-3.4 MHz
- **线数**：2 线（SCL、SDA）
- **通信**：同步半双工
- **主从**：支持多主多从

### I2C IP 供应商

| 供应商 | IP | 特性 | 典型应用 |
|--------|------|------|----------|
| Synopsys | DesignWare I2C | 主/从、DMA | 通用 |
| Cadence | Cadence I2C | 主/从、DMA | 通用 |
| ARM | ARM I2C | 主模式 | 嵌入式 |

### I2C 配置

| 配置 | 选项 | 说明 |
|------|------|------|
| 模式 | 主/从/多主 | 工作模式 |
| 速率 | 100k/400k/1M/3.4M | 时钟频率 |
| 地址 | 7-bit/10-bit | 设备地址 |
| DMA | 支持/不支持 | 数据传输 |
| FIFO | 深度 | 缓冲大小 |
| 仲裁 | 支持/不支持 | 多主仲裁 |

---

## UART IP

### UART 特性

- **速率**：9600-3M bps
- **线数**：2 线（TX、RX）
- **通信**：异步全双工
- **主从**：点对点

### UART IP 供应商

| 供应商 | IP | 特性 | 典型应用 |
|--------|------|------|----------|
| Synopsys | DesignWare UART | 全功能、DMA | 通用 |
| Cadence | Cadence UART | 全功能、DMA | 通用 |
| ARM | ARM UART | 基本功能 | 嵌入式 |

### UART 配置

| 配置 | 选项 | 说明 |
|------|------|------|
| 速率 | 9600-3M bps | 波特率 |
| 数据位 | 5-8 | 数据位数 |
| 停止位 | 1/1.5/2 | 停止位数 |
| 校验 | 无/奇/偶 | 校验方式 |
| 流控 | 无/RTS-CTS | 流量控制 |
| DMA | 支持/不支持 | 数据传输 |
| FIFO | 深度 | 缓冲大小 |

## 选型建议

### 嵌入式 MCU

- **推荐**：ARM SPI/I2C/UART
- **配置**：基本功能，无 DMA
- **理由**：面积小，功耗低

### IoT SoC

- **推荐**：Synopsys SPI/I2C/UART
- **配置**：完整功能，DMA 支持
- **理由**：功能完整，性能好

### 通用 SoC

- **推荐**：Synopsys/Cadence SPI/I2C/UART
- **配置**：完整功能，DMA 支持，多实例
- **理由**：功能完整，灵活配置

---

## 集成注意事项

### 接口匹配

- **APB 接口**：确保与 SoC 互联匹配
- **时钟域**：外设时钟与系统时钟同步
- **中断**：传输完成中断

### 验证策略

- **协议验证**：SPI/I2C/UART 协议一致性
- **性能验证**：速率和延迟测试
- **兼容性**：与外设兼容

### 物理集成

- **引脚复用**：与其他外设共享引脚
- **电源**：低功耗设计
- **DFT**：扫描链集成
