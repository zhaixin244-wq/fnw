## 设计阶段划分

### 前端设计（Frontend）

| 阶段 | 工具 | 输出 |
|------|------|------|
| 架构设计 | - | 设计规格书 |
| RTL 编码 | Verilog/SystemVerilog | RTL 代码 |
| 功能验证 | VCS/Questa/UVM | 验证报告 |
| 综合 | Design Compiler | 门级网表 |
| 形式验证 | Formality | 等价性检查报告 |
| DFT 插入 | DFT Compiler | 扫描链网表 |

### 后端设计（Backend）

| 阶段 | 工具 | 输出 |
|------|------|------|
| 布局布线 | ICC2/Innovus | 物理版图 |
| 时钟树综合 | ICC2/Innovus | 时钟树 |
| 优化 | ICC2/Innovus | 优化版图 |
| STA | PrimeTime | 时序报告 |
| DRC/LVS | Calibre | 物理验证报告 |
| 签核 | - | 签核报告 |

## 设计质量指标

### 功能正确性

- 功能验证覆盖率 > 95%
- 形式验证通过
- 门级仿真通过

### 时序收敛

- Setup 时间满足
- Hold 时间满足
- 时钟偏移 < 100ps
- 时钟抖动 < 50ps

### 功耗目标

- 动态功耗 < 预算
- 静态功耗 < 预算
- IR Drop < 10%
- EM 满足

### 物理验证

- DRC Clean
- LVS Clean
- Antenna 满足
- Density 满足
