# 芯片设计流程

> **用途**：芯片设计全流程参考，供芯片项目管理和设计时检索
> **典型应用**：所有芯片设计项目

---

## 概述

芯片设计是一个复杂的系统工程，涉及多个阶段和多种技术领域。

### 设计流程阶段

| 阶段 | 主要工作 | 输出 |
|------|----------|------|
| 规格定义 | 需求分析、架构设计 | 设计规格书 |
| RTL 设计 | 编码、仿真、验证 | RTL 代码 |
| 逻辑综合 | RTL → 门级网表 | 门级网表 |
| 物理设计 | 布局布线、时钟树 | 物理版图 |
| 签核验证 | STA、DRC/LVS | 签核报告 |
| 流片制造 | 提交 GDS、制造 | 芯片 |

---

## 阶段 1：规格定义

### 1.1 需求分析

- **功能需求**：芯片功能定义
- **性能需求**：频率、带宽、延迟
- **功耗需求**：动态功耗、静态功耗
- **面积需求**：芯片面积目标
- **接口需求**：外部接口定义

### 1.2 架构设计

- **系统架构**：模块划分、接口定义
- **数据通路**：数据流设计
- **控制通路**：控制逻辑设计
- **时钟架构**：时钟域划分
- **电源架构**：电源域划分

### 1.3 设计规格书

```
设计规格书包含：
1. 功能描述
2. 接口定义
3. 时序要求
4. 功耗预算
5. 面积预算
6. 验证计划
```

---

## 阶段 2：RTL 设计

### 2.1 RTL 编码

- **语言**：Verilog / SystemVerilog
- **风格**：可综合 RTL
- **规范**：编码规范检查

### 2.2 功能验证

- **单元验证**：模块级验证
- **集成验证**：系统级验证
- **回归验证**：持续集成验证

### 2.3 RTL 签核

- **Lint 检查**：代码风格检查
- **CDC 检查**：跨时钟域检查
- **形式验证**：属性检查

---

## 阶段 3：逻辑综合

### 3.1 综合流程

```
RTL 代码
    ↓
读入设计
    ↓
设置约束
    ↓
编译优化
    ↓
生成网表
    ↓
时序报告
```

### 3.2 综合约束

| 约束类型 | 说明 |
|----------|------|
| 时钟约束 | 时钟周期、偏移、抖动 |
| 输入延迟 | 输入信号延迟 |
| 输出延迟 | 输出信号延迟 |
| 负载 | 输出负载电容 |
| 驱动 | 输入驱动强度 |

### 3.3 综合优化

- **时序优化**：关键路径优化
- **面积优化**：逻辑共享、资源复用
- **功耗优化**：时钟门控、操作数隔离

---

## 阶段 4：DFT 插入

### 4.1 DFT 流程

```
门级网表
    ↓
扫描链插入
    ↓
BIST 插入
    ↓
边界扫描插入
    ↓
DFT 网表
```

### 4.2 DFT 组件

| 组件 | 说明 |
|------|------|
| 扫描链 | 串行测试路径 |
| BIST | 内建自测试 |
| 边界扫描 | JTAG 接口 |
| ATPG | 自动测试向量生成 |

---

## 阶段 5：物理设计

### 5.1 物理设计流程

```
门级网表 + 约束
    ↓
布局规划
    ↓
单元放置
    ↓
时钟树综合
    ↓
布线
    ↓
优化
    ↓
物理版图
```

### 5.2 布局规划

- **芯片尺寸**：面积估算
- **I/O 布局**：引脚位置
- **电源规划**：电源网络设计
- **宏单元布局**：SRAM、IP 位置

### 5.3 时钟树综合

- **时钟树结构**：H 树、鱼骨树
- **时钟偏移**：最小化偏移
- **时钟抖动**：控制抖动
- **时钟功耗**：降低功耗

### 5.4 布线

- **全局布线**：布线规划
- **详细布线**：实际布线
- **布线优化**：时序、拥塞优化

---

## 阶段 6：签核验证

### 6.1 静态时序分析（STA）

- **Setup 检查**：数据到达时间
- **Hold 检查**：数据保持时间
- **时钟偏移**：时钟到达时间差
- **OCV**：片上变化分析

### 6.2 物理验证

- **DRC**：设计规则检查
- **LVS**：版图与原理图一致性
- **ERC**：电气规则检查
- **Antenna**：天线效应检查

### 6.3 功耗分析

- **IR Drop**：电压降分析
- **EM**：电迁移分析
- **功耗估算**：动态/静态功耗

---

## 阶段 7：流片制造

### 7.1 GDS 生成

- **数据格式**：GDSII 格式
- **层次结构**：模块层次
- **数据量**：GB 级别

### 7.2 流片提交

- **工艺选择**：TSMC/Samsung/SMIC
- **MPW**：多项目晶圆
- **Full Mask**：全掩膜

### 7.3 制造流程

```
GDS
    ↓
掩膜制造
    ↓
晶圆制造
    ↓
封装测试
    ↓
芯片成品
```

---

## 设计迭代

### 时序不收敛

1. **分析瓶颈**：关键路径分析
2. **优化 RTL**：流水线、并行化
3. **调整约束**：放松非关键路径
4. **手动优化**：关键路径手动优化

### 功耗超标

1. **时钟门控**：增加时钟门控
2. **电源门控**：增加电源门控
3. **多电压域**：降低非关键模块电压
4. **操作数隔离**：减少无效翻转

### 面积超标

1. **逻辑优化**：减少逻辑量
2. **资源共享**：共享运算单元
3. **时分复用**：复用硬件资源
4. **手动优化**：关键模块手动优化

## 设计质量检查清单

### RTL 签核

- [ ] Lint 检查通过
- [ ] CDC 检查通过
- [ ] 功能验证覆盖率 > 95%
- [ ] 形式验证通过
- [ ] 门级仿真通过

### 综合签核

- [ ] 时序满足（WNS > 0）
- [ ] 面积满足预算
- [ ] 功耗满足预算
- [ ] 等价性检查通过

### 物理设计签核

- [ ] STA 满足
- [ ] DRC Clean
- [ ] LVS Clean
- [ ] IR Drop < 10%
- [ ] EM 满足
