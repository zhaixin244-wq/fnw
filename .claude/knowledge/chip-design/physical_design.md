# 物理设计

> **用途**：芯片物理设计流程参考，供后端设计时检索
> **典型应用**：所有芯片物理设计

---

## 概述

物理设计是将门级网表转换为实际版图的过程。

### 物理设计流程

```
门级网表 + 约束
    ↓
布局规划（Floorplan）
    ↓
单元放置（Placement）
    ↓
时钟树综合（CTS）
    ↓
布线（Routing）
    ↓
优化（Optimization）
    ↓
物理版图
```

## 布局规划（Floorplan）

### 布局规划内容

| 内容 | 说明 |
|------|------|
| 芯片尺寸 | 面积估算 |
| I/O 布局 | 引脚位置 |
| 电源规划 | 电源网络设计 |
| 宏单元布局 | SRAM、IP 位置 |
| 区域划分 | 电源域、时钟域 |

### 芯片尺寸估算

```
面积 = 逻辑面积 + 存储面积 + 布线通道 + 电源网络 + 填充

示例：
- 逻辑面积：1 mm²
- 存储面积：2 mm²
- 布线通道：30% 额外面积
- 电源网络：10% 额外面积
- 填充：5% 额外面积

总面积 = (1 + 2) × 1.45 = 4.35 mm²
```

### I/O 布局

```
┌─────────────────────────────────────────────┐
│  I/O 布局                                   │
│  ┌──────────────────────────────────────┐   │
│  │  North: 数据 I/O                      │   │
│  └──────────────────────────────────────┘   │
│  ┌─────┐                              ┌─────┐│
│  │West │                              │East ││
│  │控制 │                              │电源 ││
│  │I/O  │                              │I/O  ││
│  └─────┘                              └─────┘│
│  ┌──────────────────────────────────────┐   │
│  │  South: 时钟 + 电源                   │   │
│  └──────────────────────────────────────┘   │
└─────────────────────────────────────────────┘
```

## 单元放置（Placement）

### 放置流程

```
1. 全局放置
   - 粗略放置所有单元
   - 考虑时序和拥塞

2. 详细放置
   - 精确放置单元
   - 优化时序和面积

3. 放置优化
   - 时序优化
   - 拥塞优化
   - 功耗优化
```

### 放置约束

```tcl
# 区域约束
create_region region1 -rect {0 0 100 100}
add_cells_to_region region1 [get_cells u1/*]

# 位置约束
set_cell_location -coordinates {10 20} -fixed [get_cells u1/reg0]

# 禁止区域
create_blockage -rect {50 50 60 60} -type hard
```

### 放置质量指标

| 指标 | 目标 | 说明 |
|------|------|------|
| 时序 | WNS > 0 | 最差负裕量 |
| 拥塞 | < 5% | 拥塞百分比 |
| 密度 | 70-80% | 单元密度 |
| 面积 | < 预算 | 总面积 |

## 时钟树综合（CTS）

### 时钟树目标

- **最小化偏移**：时钟到达时间差最小
- **控制抖动**：时钟抖动在预算内
- **低功耗**：时钟树功耗最小
- **可布线**：时钟树可布线

### 时钟树结构

```
H 树结构：
                    clk
                     │
            ┌────────┼────────┐
            │        │        │
           clk1    clk2    clk3
            │        │        │
         ┌──┼──┐  ┌──┼──┐  ┌──┼──┐
         │     │  │     │  │     │
       clk1a clk1b clk2a clk2b clk3a clk3b

鱼骨树结构：
                    clk
                     │
         ┌───────────┼───────────┐
         │           │           │
    ─────┼───────────┼───────────┼─────
         │           │           │
       ┌─┼─┐       ┌─┼─┐       ┌─┼─┐
       │   │       │   │       │   │
      clk1 clk2   clk3 clk4   clk5 clk6
```

### 时钟树综合约束

```tcl
# 时钟树根
set_clock_tree_references -references {CLKBUF_X1 CLKBUF_X2 CLKBUF_X4}

# 时钟树目标
set_clock_tree_latency -rise 0.5 -fall 0.5
set_clock_tree_uncertainty 0.1

# 时钟树例外
set_clock_tree_exceptions -exclude [get_pins u1/clk]
```

### 时钟树质量指标

| 指标 | 目标 | 说明 |
|------|------|------|
| 偏移 | < 100 ps | 时钟到达时间差 |
| 抖动 | < 50 ps | 时钟抖动 |
| 功耗 | < 预算 | 时钟树功耗 |
| 级数 | < 10 | 时钟树级数 |

## 布线（Routing）

### 布线流程

```
1. 全局布线
   - 规划布线路径
   - 分配布线资源

2. 详细布线
   - 实际布线
   - 满足设计规则

3. 布线优化
   - 时序优化
   - DRC 修复
```

### 布线层

| 层 | 用途 | 方向 |
|------|------|------|
| M1 | 局部布线 | 水平 |
| M2 | 局部布线 | 垂直 |
| M3-M4 | 中等布线 | 水平/垂直 |
| M5-M6 | 长距离布线 | 水平/垂直 |
| M7-M8 | 电源/时钟 | 水平/垂直 |
| AP | 顶层布线 | 水平/垂直 |

### 布线约束

```tcl
# 布线层限制
set_routing_layer_min_layer M2
set_routing_layer_max_layer M6

# 非默认规则
create_non_default_rule -name ndr1 -width {M2 0.2 M3 0.2}
set_clock_routing_rules -rule ndr1

# 屏蔽线
set_shield_net VSS
```

## 优化（Optimization）

### 优化阶段

| 阶段 | 工具 | 优化内容 |
|------|------|----------|
| 放置后优化 | ICC2/Innovus | 时序、拥塞 |
| CTS 后优化 | ICC2/Innovus | 时钟偏移、时序 |
| 布线后优化 | ICC2/Innovus | 时序、DRC |

### 优化方法

| 方法 | 说明 | 应用 |
|------|------|------|
| 缓冲器插入 | 插入缓冲器 | 修复时序 |
| 单元替换 | 替换单元 | 修复时序 |
| 重布线 | 重新布线 | 修复 DRC |
| 逻辑重构 | 重构逻辑 | 修复时序 |

### 优化约束

```tcl
# 时序优化
set_optimize_timing -setup -hold

# 面积优化
set_optimize_area -effort high

# 功耗优化
set_optimize_power -clock_gating
```

## 物理验证

### DRC（设计规则检查）

```
DRC 检查内容：
1. 最小宽度
2. 最小间距
3. 最小面积
4. 密度规则
5. 天线规则
```

### LVS（版图与原理图一致性）

```
LVS 检查内容：
1. 器件匹配
2. 连接关系匹配
3. 参数匹配
4. 电源/地匹配
```

### ERC（电气规则检查）

```
ERC 检查内容：
1. 悬空引脚
2. 短路
3. 开路
4. 电源/地连接
```

## 寄生参数提取

### 提取流程

```
物理版图
    ↓
寄生参数提取工具（StarRC/Quantus）
    ↓
寄生参数网表（SPEF/DSPF）
    ↓
STA 分析
```

### 提取精度

| 精度 | 说明 | 应用 |
|------|------|------|
| 规则提取 | 基于规则 | 快速估算 |
| 场提取 | 基于场求解 | 精确提取 |
| 混合提取 | 结合两者 | 平衡速度和精度 |

## 常见问题

### 拥塞问题

1. **分析拥塞**：拥塞热图
2. **调整布局**：重新放置单元
3. **增加布线资源**：增加布线层
4. **优化逻辑**：减少布线需求

### 时序问题

1. **分析关键路径**：时序报告
2. **优化放置**：移动关键单元
3. **优化布线**：使用更快的布线层
4. **插入缓冲器**：修复时序违规

### IR Drop 问题

1. **分析 IR Drop**：IR Drop 热图
2. **优化电源网络**：增加电源条带
3. **调整布局**：分散高功耗单元
4. **降低功耗**：减少动态功耗
