# 接口协议知识库总览

> **用途**：芯片设计常用接口协议参考文档，供 chip-arch agent 在架构设计时快速检索
> **总计**：16 个协议文档，覆盖 5 大类别

---

## 协议分类索引

### 1. AMBA 总线族（5 个）

| 协议 | 文件 | 速率/带宽 | 定位 | 典型应用 |
|------|------|-----------|------|----------|
| AXI4 | [axi4.md](axi4.md) | 128-bit@250MHz = 4GB/s | 高性能系统总线 | DDR 控制器、DMA、高速外设 |
| AXI4-Lite | [axi4_lite.md](axi4_lite.md) | 32-bit 单拍 | 低速寄存器访问 | CSR、中断控制器、外设配置 |
| AXI4-Stream | [axi4_stream.md](axi4_stream.md) | 与 TDATA 宽度成正比 | 流式数据传输 | 视频流、DSP 链路、包处理 |
| AHB | [ahb.md](ahb.md) | 32/64-bit 流水线 | 高性能系统总线（AXI 前代） | CPU 总线、SRAM 控制器 |
| APB | [apb.md](apb.md) | 32-bit 单拍 | 外设低功耗总线 | UART/SPI/I2C 寄存器 |

**选型指南**：
- CPU ↔ 内存/DMA → AXI4
- CPU ↔ 外设寄存器 → AXI4-Lite 或 APB
- 数据流（视频/网络） → AXI4-Stream
- 旧 IP 集成 → AHB
- 低功耗外设 → APB（通过 Bridge 连接 AXI/AHB）

### 2. 串行外设接口（4 个）

| 协议 | 文件 | 速率 | 线数 | 通信方式 | 典型应用 |
|------|------|------|------|----------|----------|
| SPI | [spi.md](spi.md) | 1-100+ MHz | 4 线 | 同步全双工主从 | Flash、传感器、ADC/DAC |
| I2C | [i2c.md](i2c.md) | 100k-3.4 MHz | 2 线 | 同步半双工多主多从 | EEPROM、PMIC、传感器 |
| UART | [uart.md](uart.md) | 9600-3M bps | 2 线 | 异步全双工点对点 | 调试串口、GPS、蓝牙 |
| CAN | [can.md](can.md) | 1 Mbps (FD: 8 Mbps) | 2 线(差分) | 同步多主仲裁 | 汽车 ECU、工业控制 |

**选型指南**：
- 高速外设 + 简单协议 → SPI
- 多设备 + 省引脚 → I2C
- 调试/人机交互 → UART
- 汽车/工业环境 → CAN

### 3. 高速串行接口（3 个）

| 协议 | 文件 | 速率 | 拓扑 | 典型应用 |
|------|------|------|------|----------|
| PCIe | [pcie.md](pcie.md) | Gen5: 32GT/s, Gen6: 64GT/s (PAM4) | 点对点 | NVMe SSD、GPU、网卡 |
| USB | [usb.md](usb.md) | HS: 480Mbps, SS: 20Gbps | 主从树形 | 外设、充电、数据传输 |
| MIPI | [mipi.md](mipi.md) | D-PHY: 2.5Gbps/lane | 点对点 | 摄像头(CSI)、显示屏(DSI) |

**选型指南**：
- PC/服务器高速互联 → PCIe
- 消费电子外设 → USB
- 移动设备摄像头/显示 → MIPI

## 协议对比速查

### AMBA 总线族对比

| 特性 | AXI4 | AXI4-Lite | AXI4-Stream | AHB | APB |
|------|------|-----------|-------------|-----|-----|
| 突发传输 | 支持 (256 拍) | 不支持 | N/A | 支持 (16 拍) | 不支持 |
| Outstanding | 支持 | 不支持 | N/A | 不支持 | 不支持 |
| 乱序完成 | 支持 (同 ID) | 不支持 | N/A | 不支持 | 不支持 |
| 流控 | Valid/Ready | Valid/Ready | Valid/Ready | HREADY | PREADY |
| 地址空间 | 有 | 有 | 无 | 有 | 有 |
| 通道数 | 5 | 5 | 1 | 1 (流水线) | 1 |
| 信号数 | ~50 | ~20 | 9+ | ~30 | ~11 |
| 适用场景 | 高带宽 | 寄存器 | 流数据 | CPU 总线 | 外设 |

### 串行接口对比

| 特性 | SPI | I2C | UART | CAN |
|------|-----|-----|------|-----|
| 线数 | 4 | 2 | 2 | 2(差分) |
| 时钟线 | 有 (SCLK) | 有 (SCL) | 无 | 无 |
| 全双工 | 是 | 否 | 是 | 否 |
| 多设备 | 片选扩展 | 地址寻址 | 点对点 | 多主仲裁 |
| 最高速率 | 100+ MHz | 3.4 MHz | 3 Mbps | 8 Mbps (FD) |
| 最大设备数 | 受 CS 限制 | 127 (7-bit) | 2 | 理论无限制 |
| 错误检测 | 无 | ACK/NACK | Parity/Frame | CRC/5 种错误 |

## 文档格式说明

每个协议文档包含以下标准章节：

| 章节 | 内容 |
|------|------|
| 协议概述 | 特性、定位、规范版本、典型应用 |
| 信号定义 | 信号表：方向、位宽、功能说明 |
| 事务流程 | 读写时序 ASCII 波形图 |
| 关键机制 | 仲裁、流控、错误处理、编码等 |
| 与其他协议对比 | 特性对比表 |
| 设计注意事项 | RTL 实现要点、常见陷阱 |

**配合 chip-arch agent 使用**：在生成 FS/微架构文档时，agent 会检索本目录下的协议文档作为接口设计参考。
