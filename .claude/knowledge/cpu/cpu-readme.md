## ISA 选型速查

| 特性 | RISC-V | ARM | MIPS |
|------|--------|-----|------|
| 授权 | 开源免费 | 商业授权 | 商业授权（部分开放） |
| 生态 | 快速成长 | 最成熟 | 成熟但萎缩 |
| 扩展性 | 模块化（I/M/A/F/D/C/V） | 固定 ISA + 扩展 | 固定 ISA |
| 适用场景 | 定制化、AI、IoT | 通用计算 | 网络、嵌入式 |
| 工具链 | GCC/LLVM | GCC/LLVM/ARM CC | GCC |
| 典型频率 | 100MHz~2GHz | 100MHz~3GHz+ | 100MHz~1GHz |

## 处理器微架构层次

```
┌─────────────────────────────────────────────────────────┐
│                    应用处理器 (AP)                        │
│        Out-of-Order / Superscalar / SMT                 │
├─────────────────────────────────────────────────────────┤
│                    核心微架构                              │
│  分支预测 → 取指 → 译码 → 重命名 → 发射 → 执行 → 提交    │
├─────────────────────────────────────────────────────────┤
│                    缓存层次                                │
│  L1 I-Cache (32KB) │ L1 D-Cache (32KB) │ L2 (256KB-1MB)│
├─────────────────────────────────────────────────────────┤
│                    一致性互联                              │
│        AXI/CHI/TileLink — Snoop/Directory               │
├─────────────────────────────────────────────────────────┤
│                    L3 Cache / LLC                        │
│            共享缓存，Directory 一致性                      │
├─────────────────────────────────────────────────────────┤
│                    内存控制器                              │
│            DDR4/DDR5/LPDDR5                              │
└─────────────────────────────────────────────────────────┘
```

## 典型处理器产品与架构需求

| 处理器类型 | 架构需求 |
|-----------|---------|
| **嵌入式 MCU** | 单发射顺序执行、无 Cache 或小 Cache、低功耗、小面积 |
| **IoT 处理器** | 超低功耗、安全扩展（TrustZone）、无线接口 |
| **应用处理器（手机）** | 多核 OoO、大 Cache、NEON/SVE、GPU 协同 |
| **服务器 CPU** | 高核数 OoO、大 L3、SMT、RAS、安全扩展 |
| **AI 加速器** | 向量扩展（V/SVE）、大 SRAM、高带宽内存接口 |
| **网络处理器** | 多线程、硬件加速器、包处理流水线 |

## 文档格式说明

每个架构文档包含以下标准章节：

| 章节 | 内容 |
|------|------|
| 架构概述 | ISA 特性、版本、典型实现 |
| 指令集/组件 | 指令格式、寄存器、关键机制 |
| 流水线/时序 | 流水线级数、冒险处理、时序图 |
| 扩展/配置 | 可选扩展、配置参数 |
| 设计注意事项 | RTL 实现要点、面积/功耗/性能权衡 |

**配合 chip-arch agent 使用**：在生成 CPU 子系统 FS/微架构文档时，agent 会检索本目录下的文档作为设计参考。
