# CAN

多主广播总线，非破坏性仲裁，广泛用于汽车电子。

## 基本信息

| 字段 | 内容 |
|------|------|
| **类型** | 串行通信总线 |
| **版本** | ISO 11898（数据链路层+物理层） |
| **来源** | Bosch |

## 核心特征

- **多主广播**：所有节点可同时收发，广播式通信
- **差分信号**：CAN_H/CAN_L，ISO 11898-2（高速）/ ISO 11898-3（低速容错）
- **非破坏性仲裁**：CSMA/CD+AMP，基于消息 ID 优先级
- **5 种错误检测**：位错误/填充错误/CRC 错误/格式错误/ACK 错误
- **3 级错误状态**：Error Active → Error Passive → Bus Off
- **CAN-FD 扩展**：数据段最高 8 Mbps，最大 64 字节载荷

## 关键信号

| 信号 | 说明 |
|------|------|
| `CAN_H` | 差分信号高（显性时 ~3.5V） |
| `CAN_L` | 差分信号低（显性时 ~1.5V） |

## 关键参数

| 参数 | Classical CAN | CAN-FD |
|------|---------------|--------|
| 仲裁段速率 | 最高 1 Mbps | 最高 1 Mbps |
| 数据段速率 | 最高 1 Mbps | 最高 8 Mbps |
| 数据长度 | 0~8 bytes | 0~64 bytes |
| ID 长度 | 11 bit（标准）/ 29 bit（扩展） | 同左 |
| CRC | CRC-15 | CRC-17/CRC-21 |
| 终端电阻 | 120 ohm（总线两端） | 同左 |

## 典型应用

- 汽车 ECU 通信（动力/底盘/车身）
- 工业控制总线
- 医疗设备
- 航空航天

## 与其他协议的关系

- **CAN-FD**：CAN 扩展版，更高数据速率和更长载荷
- **LIN**：汽车低速总线，CAN 的补充
- **Ethernet**：车载以太网，CAN 的高速替代方案

## 设计要点

- 显性（逻辑 0）覆盖隐性（逻辑 1）——仲裁物理基础
- 仲裁阶段：发送隐性但读到显性则退出仲裁
- 位填充：连续 5 位相同后插入 1 位反转
- TEC/REC 计数器管理错误状态转换
- 120 ohm 终端电阻必须在总线两端
