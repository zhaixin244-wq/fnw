# sta_advanced

> STA 高级概念，涵盖 AOCV/SOCV、信号完整性、多角分析、高级时序优化

## 基本信息

| 字段 | 值 |
|------|----|
| 类型 | 方法论 |
| 来源 | .claude/knowledge/chip-design/sta_advanced.md |

## 核心特性
- AOCV（高级片上变化）：考虑路径深度和物理距离对时序变化的影响
- SOCV（统计片上变化）：统计方法分析时序变化，3σ/4σ 裕量
- 信号完整性（SI）：串扰分析（功能错误/时序影响/毛刺）
- 时钟树分析：偏移/抖动/转换时间/功耗
- 多角分析：PVT 角（SS/FF/TT/SF/FS）
- 高级优化：重定时/逻辑复制/引脚交换/单元替换

## 关键参数

| 指标 | 目标 | 说明 |
|------|------|------|
| 时钟偏移 | < 100ps | 最大-最小时钟到达时间差 |
| 时钟抖动 | < 50ps | 时钟周期变化 |
| 转换时间 | < 200ps | 时钟信号转换时间 |
| AOCV 系数 | 路径相关 | 深度 1:1.0, 深度 5:0.95, 深度 10:0.90 |

## 典型应用场景
- 高性能芯片时序分析
- 多角签核验证
- 串扰分析与优化

## 与其他实体的关系
- **sta_basics**：基础概念延伸
- **signoff**：签核必须覆盖所有 PVT 角
- **physical_design**：时钟树优化与布线优化

## 设计注意事项
- AOCV/SOCV 可放松非关键路径裕量，但关键路径仍需严格检查
- SI 优化：增加间距/插入缓冲器/使用屏蔽线
- 多角分析必须覆盖 SS（Setup 最差）和 FF（Hold 最差）
