# USB

通用串行总线，主从架构高速外设接口。

## 基本信息

| 字段 | 内容 |
|------|------|
| **类型** | 串行总线 |
| **版本** | USB 2.0 / USB 3.x / USB4 |
| **来源** | USB-IF |

## 核心特征

- **主从架构**：Host - Hub - Device，星形/树形拓扑
- **4 种传输类型**：Control/Bulk/Interrupt/Isochronous
- **自同步编码**：NRZI+位填充（2.0）/ 8b10b（3.x）/ 128b132b（USB4）
- **即插即用**：设备枚举、热插拔
- **供电**：VBUS 5V，标准 500mA，BC 1.2 扩展至 2A+
- **速率演进**：1.5 Mbps → 480 Mbps → 5 Gbps → 20/40 Gbps

## 关键信号

| 信号 | USB 2.0 | USB 3.x | 说明 |
|------|---------|---------|------|
| `VBUS` | 有 | 有 | +5V 电源 |
| `D+/D-` | 有 | 兼容 | USB 2.0 差分数据 |
| `SSTX+/-` | - | 有 | SuperSpeed 发送差分对 |
| `SSRX+/-` | - | 有 | SuperSpeed 接收差分对 |

## 关键参数

| 参数 | LS/FS/HS | SS/SS+ | USB4 |
|------|----------|--------|------|
| 最高速率 | 480 Mbps | 10 Gbps | 40 Gbps |
| 编码 | NRZI+位填充 | 8b/10b | 128b/132b |
| 双工 | 半双工 | 全双工 | 全双工 |
| 最大电缆长度 | 3~5 m | 1~3 m | 0.8 m |

## 典型应用

- 存储设备（U盘、移动硬盘）
- HID（键盘、鼠标）
- 摄像头、音频设备
- Type-C 生态

## 与其他协议的关系

- **PCIe**：USB 主机控制器通过 PCIe 连接 CPU
- **MIPI**：USB 用于外部连接，MIPI 用于内部摄像头/显示
- **Thunderbolt**：融合 PCIe + USB4

## 设计要点

- USB 2.0 连接检测：Device 端 D+/D- 上拉 1.5kΩ
- HS 协商：FS 上拉 → Chirp 握手 → 切换 HS
- 数据切换：DATA0/DATA1 交替，保证传输完整性
- 4 种传输类型有不同带宽和延迟特性
- UTMI/ULPI 为 PHY 接口标准
