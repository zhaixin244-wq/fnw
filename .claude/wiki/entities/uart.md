# UART

异步串行通信接口，点对点全双工。

## 基本信息

| 字段 | 内容 |
|------|------|
| **类型** | 异步串行接口 |
| **版本** | 事实标准（RS-232/RS-422/RS-485 物理层） |
| **来源** | 通用标准 |

## 核心特征

- **2 线制**：TX（发送）+ RX（接收），全双工
- **异步通信**：无时钟信号线，收发通过波特率同步
- **点对点**：一对一连接，无寻址机制
- **可配置帧格式**：5~8 数据位、奇偶校验、1/1.5/2 停止位
- **硬件流控**：RTS/CTS 可选

## 关键信号

| 信号 | 方向 | 说明 |
|------|------|------|
| `TX` | 输出 | 发送数据线 |
| `RX` | 输入 | 接收数据线 |
| `RTS` | 输出 | Request To Send（可选，低有效） |
| `CTS` | 输入 | Clear To Send（可选，低有效） |

## 关键参数

| 参数 | 范围 | 说明 |
|------|------|------|
| 波特率 | 300~3000000 bps | 常用 9600/115200/921600 |
| 数据位 | 5/6/7/8 bit | 通常 8 bit |
| 停止位 | 1 / 1.5 / 2 bit | 通常 1 bit |
| 校验 | None/Even/Odd | 可选 |
| 过采样 | 16x | 接收端采样时钟 |

## 典型应用

- 调试串口（Console）
- 板间通信
- GPS/蓝牙模块
- 嵌入式控制台

## 与其他协议的关系

- **SPI**：同步 4 线全双工，有 SCLK 时钟
- **I2C**：同步 2 线半双工，有 SCL 时钟和地址
- UART 是最简单的异步串行协议

## 设计要点

- 空闲时 TX/RX 为高电平
- 起始位（低）→ 数据位（LSB 先发）→ 校验位 → 停止位（高）
- 接收端 16x 过采样定位数据位中心
- 波特率误差容限：通常 ≤ 3%
- FIFO 缓冲可减少 CPU 中断频率
