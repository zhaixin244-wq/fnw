# I2C

双线制同步串行总线，多主多从半双工通信。

## 基本信息

| 字段 | 内容 |
|------|------|
| **类型** | 串行总线 |
| **版本** | NXP I2C-bus specification v6.0 (UM10204) |
| **来源** | NXP（原 Philips） |

## 核心特征

- **2 线制**：SCL（时钟）+ SDA（数据），开漏输出需上拉电阻
- **半双工**：同一时刻数据单向传输
- **多主多从**：Multi-Master Multi-Slave 拓扑
- **无损仲裁**：基于线与（Wired-AND）逻辑
- **7/10 bit 寻址**：最多 112（7-bit）或 1024（10-bit）从设备
- **时钟拉伸**：从设备可拉低 SCL 暂停通信

## 关键信号

| 信号 | 类型 | 说明 |
|------|------|------|
| `SCL` | 开漏双向 | 时钟信号，Master 产生；从设备可 Clock Stretching |
| `SDA` | 开漏双向 | 数据信号，传输地址/数据/ACK/NACK |

## 关键参数

| 参数 | 范围 | 说明 |
|------|------|------|
| Standard Mode | 100 kHz | 标准速率 |
| Fast Mode | 400 kHz | 快速模式 |
| Fast Mode Plus | 1 MHz | 增强快速模式 |
| High Speed Mode | 3.4 MHz | 高速模式 |
| 最大总线电容 | 400 pF | 限制设备数和走线长度 |
| 上拉电阻 | 典型 4.7 kΩ | 取决于速率和电容 |

## 典型应用

- 传感器（温度、压力、加速度）
- EEPROM/Flash 配置
- PMIC 电源管理
- 板级芯片间通信
- SMBus 扩展（智能电池）

## 与其他协议的关系

- **SPI**：4 线全双工，速率更高，无地址/ACK
- **UART**：异步点对点，无时钟线
- **MIPI I3C**：I2C 升级版，更高速率，混合拓扑

## 设计要点

- 开漏输出需外部上拉电阻
- START：SCL 高时 SDA 下降沿；STOP：SCL 高时 SDA 上升沿
- ACK：第 9 个时钟 SDA 拉低表示应答
- 仲裁阶段：SDA 线与逻辑，发送隐性（1）但读到显性（0）则退出
- 保留地址 0000 000 和 1111 1xx 有特殊用途
