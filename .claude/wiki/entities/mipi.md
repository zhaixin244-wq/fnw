# MIPI

移动设备摄像头/显示屏高速串行接口标准族。

## 基本信息

| 字段 | 内容 |
|------|------|
| **类型** | 高速串行接口族 |
| **版本** | CSI-2 v3.0 / DSI v2.0 / D-PHY v3.0 / C-PHY v2.1 |
| **来源** | MIPI Alliance |

## 核心特征

- **低引脚数**：串行差分传输替代并行总线
- **双模物理层**：LP（低功耗）模式用于控制，HS（高速）模式用于数据
- **多 Lane 并行**：1 对时钟 + 1~4 对数据 Lane
- **CSI-2**：摄像头图像数据传输（Camera → SoC）
- **DSI**：显示屏图像/命令传输（SoC → Display）
- **虚拟通道**：VC0~VC3，支持多路复用

## 关键信号（D-PHY）

| 信号 | 方向 | 说明 |
|------|------|------|
| `CLK_P/CLK_N` | TX→RX | 差分时钟 Lane |
| `D0_P/D0_N` | 双向 | 数据 Lane 0（必需） |
| `D1_P/D1_N` | 双向 | 数据 Lane 1~3（可选） |

## 关键参数

| 参数 | D-PHY | C-PHY |
|------|-------|-------|
| 信号线 | 2 wire（差分对） | 3 wire（7-Phase） |
| 最高速率 | 2.5 Gbps/lane | ~5.7 Gbps/trio |
| LP 模式 | 10 Mbps | 有 |
| HS 模式 | 差分 | 3 线 7-Phase 编码 |
| 最大 Lane | 1 clock + 4 data | 无独立时钟 |

## 典型应用

- 手机/平板摄像头接口
- 平板/手机显示屏接口
- 车载视觉系统
- AR/VR 显示

## 与其他协议的关系

- **USB**：USB 用于外部连接，MIPI 用于设备内部
- **I3C**：MIPI 控制总线（I2C 升级版）
- CSI-2 和 DSI 共享相同的协议层包格式

## 设计要点

- LP 模式：单端信号，用于控制命令和初始化
- HS 模式：差分信号，用于高速数据传输
- CSI-2 包格式：Short Packet（帧同步等）+ Long Packet（像素数据）
- DSI 命令：DCS（Display Command Set）标准命令集
- Virtual Channel：2 bit VC 支持最多 4 路复用
