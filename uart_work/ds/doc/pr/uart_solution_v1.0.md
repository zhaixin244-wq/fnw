# UART 模块架构方案

> 文档编号：FNW-PR-UART-SOL-v1.0 | 日期：2026-04-27 | 作者：AI Agent

---

## 1. 方案概述

基于需求汇总，采用经典 UART 架构：APB 寄存器接口 + 小数波特率发生器 + TX/RX FIFO + 发送/接收状态机。单时钟域设计，异步复位同步释放。

---

## 2. 顶层架构

```
                    APB Bus
                       │
                 ┌─────┴─────┐
                 │  APB Slave │
                 │  (寄存器)   │
                 └─────┬─────┘
                       │ reg_bus
          ┌────────────┼────────────┐
          │            │            │
    ┌─────┴─────┐ ┌───┴────┐ ┌────┴─────┐
    │ 波特率发生器 │ │  TX    │ │  RX      │
    │ (BAUD_GEN) │ │ (发送)  │ │ (接收)   │
    └─────┬─────┘ └───┬────┘ └────┬─────┘
          │            │           │
          │ baud_tick  │ txd       │ rxd
          │            │           │
          │       ┌────┴────┐ ┌───┴────┐
          │       │ TX FIFO │ │ RX FIFO │
          │       └────┬────┘ └───┬────┘
          │            │           │
          │            │ tx_irq    │ rx_irq
          │            │           │
          └────────────┴───────────┘
                       │
                   interrupt
```

## 3. 子模块划分

| 子模块 | 功能 | 关键接口 |
|--------|------|----------|
| `uart_reg_mod` | APB 从接口 + 寄存器读写 | PADDR, PWDATA, PRDATA, PSEL, PENABLE, PREADY |
| `uart_baud_gen` | 小数波特率发生器 | baud_div_int, baud_div_frac, baud_tick_16x |
| `uart_tx` | 发送状态机 + 移位寄存器 | tx_data, tx_start, tx_done, txd |
| `uart_rx` | 接收状态机 + 过采样 + 移位寄存器 | rxd, rx_data, rx_valid, rx_err |
| `uart_fifo` | 参数化同步 FIFO | wr_en, wr_data, rd_en, rd_data, full, empty |
| `uart_ctrl` | 中断逻辑 + 模式控制 + 状态聚合 | irq, loopback_en, flow_ctrl |
| `uart_top` | 顶层集成（仅连线） | APB + UART 外部信号 |

## 4. 关键设计决策

### 4.1 ADR-001: 波特率生成采用小数分频

**背景**：整数分频在高频时钟下波特率精度不够（如 50MHz 时钟 / 16 / 115200 = 27.127，整数分频为 27，误差 0.47%）。

**决策**：采用 16 位整数 + 4 位小数的分频方案，每个 baud_tick_16x 周期累加小数部分，溢出时多计一拍。

**后果**：波特率误差 < 0.1%（@50MHz，115200bps），满足 UART 规范要求的 < 2% 误差。

### 4.2 ADR-002: FIFO 深度 16 字节

**背景**：深 FIFO 可减少 CPU 中断频率，但增加面积和延迟。

**决策**：TX/RX FIFO 深度均为 16 字节，使用寄存器阵列实现。

**后果**：面积约 0.5 kGates，满足低速外设场景。后续可通过参数化扩展。

### 4.3 ADR-003: 16x 过采样

**背景**：过采样倍数影响接收精度和抗噪声能力。

**决策**：默认 16x 过采样，可配置为 8x。在 16x 模式下，中间 3 个采样点取多数判决。

**后果**：精度提升，但波特率时钟需求增加。16x 过采样时 baud_tick_16x = 系统时钟 / 波特率。

### 4.4 ADR-004: 子模块 FIFO 独立文件

**背景**：FIFO 是通用 CBB，可能被多个模块复用。

**决策**：`uart_fifo` 作为独立子模块，参数化设计（WIDTH, DEPTH），可被其他模块直接例化。

**后果**：代码复用性好，FIFO 可独立验证。

## 5. 接口定义

### 5.1 APB 从接口

| 信号 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| `pclk` | I | 1 | APB 时钟 |
| `presetn` | I | 1 | APB 复位（低有效） |
| `paddr` | I | ADDR_WIDTH | APB 地址 |
| `psel` | I | 1 | APB 选择 |
| `penable` | I | 1 | APB 使能 |
| `pwrite` | I | 1 | APB 写使能 |
| `pwdata` | I | 32 | APB 写数据 |
| `prdata` | O | 32 | APB 读数据 |
| `pready` | O | 1 | APB 就绪 |
| `pslverr` | O | 1 | APB 错误 |

### 5.2 UART 外部接口

| 信号 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| `txd` | O | 1 | 发送数据线 |
| `rxd` | I | 1 | 接收数据线 |
| `rts_n` | O | 1 | 请求发送（低有效） |
| `cts_n` | I | 1 | 清除发送（低有效） |

### 5.3 中断与控制

| 信号 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| `irq` | O | 1 | 中断输出 |
