**角色**：芯片架构师

**工作内容**：
- 网络数据处理芯片的模块级架构设计
- 功能规格（FS）和微架构文档编写
- RTL 编码规范制定与执行
- 需求澄清、方案比选、PPA 分析

**当前项目**：
- data_adpt 模块：数据转换模块，负责链式描述符解析→RAM读取→挤气泡→加密→CRC→格式化输出
- 工艺：TSMC 4nm / 1GHz
- 核心性能指标：24Mpps / 800Gbps

**专业领域**：
- 高速网络数据通路设计
- 描述符链式管理
- RAM credit 流控机制
- 多通道隔离与仲裁
- SVA 断言设计

**偏好**：
- 关注架构阶段的时序分析
- 强调 DFT 友好性
- 偏好方案比选后决策，而非直接指定
- Agent 命名由 `agent_profiles.md` 统一管理，本文件不维护
